# stevens-agentic-skills

Steven Peterson's personal Claude Code plugin marketplace.

## Install
```
/plugin marketplace add SPetersonNICE/stevens-agentic-skills
```

## Plugins

### `sdd-setup` — Devcontainer project setup

| Skill | Description |
|-------|-------------|
| `create-devcontainer-project` | Scaffold a new project with a fully configured VS Code devcontainer |
| `add-claudecode-to-project` | Add the Claude Code CLI and VS Code extension to an existing devcontainer |
| `add-beads-to-project` | Add the Beads (`bd`) graph-based issue tracker to a devcontainer |
| `add-openspec-to-project` | Add the `@fission-ai/openspec` CLI to a devcontainer |

### `sdd-workflow` — OpenSpec/Beads development workflow

| Skill | Description |
|-------|-------------|
| `distill-workflow` | Extract a reusable prompt, skill, or agent from a past conversation |
| `generate-roadmap` | Research a project's actual state and generate a phased epic roadmap |
| `generate-spec-beads` | Convert an OpenSpec change into a dependency-wired Beads task graph |
| `implement-beads` | Drive implementation of a Beads-tracked body of work |
| `spec-completion-auditor` | Audit drift between closed Beads issues and open OpenSpec tasks |
